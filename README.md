
# StarBux

A quick Starbucks clone built using the Fresh framework


## Run Locally

Clone the project

```bash
  git clone https://github.com/pdcolandrea/bux-clone
```

Go to the project directory

```bash
  cd my-project
```

Start the server

```bash
  deno task start
```
